# DNS Proxy

[English](README_EN.md) | 中文

一个轻量级的 DNS 代理服务器，支持 DNS (UDP/TCP)、DoT (DNS over TLS) 和 DoH (DNS over HTTPS) 协议，内置 Web 管理界面。

## 功能特性

- **多协议支持**：DNS (UDP/TCP)、DoT (TLS 853)、DoH (HTTPS)
- **域名规则**：支持精确匹配和子域全量匹配，可指定不同上游服务器
- **本地记录**：支持自定义 A、AAAA、CNAME、TXT 记录，本地优先响应
- **Web 管理**：基于 HTTPS 的管理面板，可视化配置和实时状态监控
- **安全认证**：AES 加密存储管理员账号密码，支持在线修改
- **自签名证书**：内置 CA 证书生成，开箱即用

## 快速开始

### 编译

```bash
go build -o dns-proxy.exe ./cmd/dns-proxy
```

### 启动

```bash
# 默认配置
.\dns-proxy.exe

# 指定参数
.\dns-proxy.exe -dns 0.0.0.0:15353 -dot 0.0.0.0:1853 -doh 0.0.0.0:1443 -http 0.0.0.0:8443
```

### 默认账号

- 用户名：`admin`
- 密码：`123456`

> ⚠️ 首次登录后请立即在「账号 · 设置」页面修改密码

## 命令行参数

| 参数 | 环境变量 | 默认值 | 说明 |
|------|----------|--------|------|
| `-dns` | `DP_DNS_ADDR` | `0.0.0.0:15353` | DNS 服务监听地址 (UDP/TCP) |
| `-dot` | `DP_DOT_ADDR` | `0.0.0.0:1853` | DoT 服务监听地址 (TLS) |
| `-doh` | `DP_DOH_ADDR` | `0.0.0.0:1443` | DoH 服务监听地址 (HTTPS) |
| `-http` | `DP_HTTP_ADDR` | `0.0.0.0:8443` | Web 管理界面监听地址 |
| `-data` | `DP_DATA_DIR` | `./data` | 数据目录（数据库和缓存） |
| `-cert-dir` | `DP_CERT_DIR` | `./certs` | 证书目录 |
| `-cert-file` | `DP_CERT_FILE` | - | HTTPS 证书文件路径（覆盖 cert-dir） |
| `-key-file` | `DP_KEY_FILE` | - | HTTPS 私钥文件路径（覆盖 cert-dir） |
| `-doh-host` | `DP_DOH_HOST` | - | DoH 域名（添加到证书 SAN） |
| `-default-upstream` | `DP_DEFAULT_UPSTREAM` | `8.8.8.8:53` | 默认上游 DNS 服务器 |
| `-dev-cert` | `DP_DEV_CERT` | `false` | 每次启动重新生成自签名证书 |

## Web 管理界面

启动后访问：`https://<本机IP>:8443/`

### 功能模块

1. **规则 · 上游**
   - 添加/编辑/删除域名匹配规则
   - 支持精确匹配和子域全量匹配
   - 可指定每个规则的上游服务器
   - 一键刷新缓存

2. **本地 · 记录**
   - 添加本地 DNS 记录（A、AAAA、CNAME、TXT）
   - 支持精确/后缀匹配模式
   - 设置 TTL 和备注

3. **账号 · 设置**
   - 修改管理员用户名和密码
   - 需要验证当前账号密码

## API 接口

所有接口需要 HTTP Basic Auth 认证。

### 规则管理

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/rules` | 获取所有规则 |
| POST | `/api/rules` | 创建规则 |
| GET | `/api/rules/{id}` | 获取单个规则 |
| PUT | `/api/rules/{id}` | 更新规则 |
| DELETE | `/api/rules/{id}` | 删除规则 |
| POST | `/api/rules/reload` | 刷新缓存 |

### 本地记录管理

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/hosts` | 获取所有本地记录 |
| POST | `/api/hosts` | 创建本地记录 |
| GET | `/api/hosts/{id}` | 获取单个记录 |
| PUT | `/api/hosts/{id}` | 更新记录 |
| DELETE | `/api/hosts/{id}` | 删除记录 |

### 系统管理

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/status` | 系统状态 |
| GET | `/api/config` | 当前配置 |
| PUT | `/api/auth` | 修改账号密码 |

### DoH 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET/POST | `/dns-query` | RFC 8484 DoH 协议 |
